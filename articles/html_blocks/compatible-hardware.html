A Comprehensive List of FTMS devices for Indoor Cycling

BullWatt is compatible with any smart trainer or exercise bike that uses the standard Bluetooth FTMS protocol and supports "Controllable" Bluetooth for setting a target power (ERG mode).

The app provides a specific list of known supported devices, which includes:

  • Domyos Challenge Bike
  • Zwift Hub
  • SUITO
  • Saris
  • Wahoo KICKR CORE
